# Site iPhone da Manu 💖

Esse projeto imita uma tela de iPhone com aplicativos que abrem declarações.

## Como editar

1. Abra o arquivo `script.js`.
2. Procure por `EDITE AQUI`.
3. Troque os textos pelas suas declarações.
4. Para colocar fotos:
   - Coloque as imagens na pasta `imagens`.
   - Use os nomes:
     - foto1.jpg
     - foto2.jpg
     - foto3.jpg
     - foto4.jpg
   - Ou edite os caminhos no `script.js`.

## Senha do app secreto

No `script.js`, altere:

```js
const senhaSecreta = "2205";
```

## Links

No `script.js`, altere:

```js
const linkMusica = "https://www.youtube.com/";
const linkVideo = "https://www.youtube.com/";
```

## Como abrir

Clique duas vezes no arquivo `index.html`.
